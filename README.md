# Usage
首先需要将奇趣营服务端，部署到tomcat，参考这里[奇趣营客户端](https://github.com/liuling07/QiQuYingServer)，
然后将commons包下面的RcpUri里面的BASEPATH路径的ip换为服务端部署后的ip即可。
分享功能可以用，这是代码里面我把app key和secret key删了，你们需要自己到第三方平台申请，奇趣营支持的分享平台有
微信、QQ、腾讯微博和新浪微博。
